# เริ่มต้นใช้งาน Novel Writer Skills อย่างรวดเร็ว

## เริ่มต้นใช้งานใน 5 นาที

### 1. ติดตั้ง (1 นาที)

```bash
npm install -g novel-writer-skills
```

### 2. สร้างโปรเจกต์ (30 วินาที)

```bash
novelwrite init my-novel
cd my-novel
```

### 3. เริ่มต้นใน Claude Code (3 นาที)

1. เปิดโฟลเดอร์ `my-novel` ใน Claude Code
2. พิมพ์คำสั่งแรก:

```
/constitution
```

3. ทำตามคำแนะนำของ AI และตอบคำถาม
4. เมื่อเสร็จแล้ว ดำเนินการต่อด้วย:

```
/specify
/clarify
/plan
/tasks
/write
```

## เพียงเท่านี้ก็เรียบร้อย!

### Skills ที่เปิดใช้งานอัตโนมัติ

คุณไม่ต้องทำอะไรเลย Skills ต่อไปนี้จะทำงานโดยอัตโนมัติ:

- 🎨 **ความรู้ประเภทเรื่อง**: เมื่อพูดถึง "โรแมนซ์"/"ลึกลับ"/"แฟนตาซี" จะเปิดใช้งานโดยอัตโนมัติ
- ✍️ **เทคนิคการเขียน**: นำไปใช้โดยอัตโนมัติเมื่อเขียนบทสนทนาหรือฉาก
- 🔍 **การตรวจสอบคุณภาพ**: ตรวจสอบเบื้องหลังระหว่างกระบวนการเขียน

### ขั้นตอนแนะนำสำหรับวันแรก

**ระยะเวลา: 2-3 ชั่วโมง**

```
09:00 - 09:20  /constitution  (หลักการเขียน)
09:20 - 10:00  /specify       (ข้อกำหนดของเรื่อง)
10:00 - 10:15  /clarify       (ชี้แจงประเด็นที่ไม่ชัดเจน)
10:15 - 11:00  /plan          (แผนการเขียน)
11:00 - 11:30  /tasks         (แบ่งรายการงาน)
11:30 - 12:00  /write         (เริ่มเขียนบทที่ 1)
```

## คำถามที่พบบ่อยสำหรับผู้เริ่มต้น

**ถาม: ต้องดำเนินการตามลำดับที่กำหนดเสมอไปหรือไม่?**
ตอบ: แนะนำให้ทำตามลำดับ แต่ขั้นตอนขั้นต่ำคือ: `/constitution` → `/specify` → `/write`

**ถาม: Skills จะเปิดใช้งานโดยอัตโนมัติจริงหรือไม่?**
ตอบ: ใช่! เพียงแค่พูดถึงเนื้อหาที่เกี่ยวข้อง (โรแมนซ์ ลึกลับ ฯลฯ) Skills ก็จะทำงานโดยอัตโนมัติ

**ถาม: ฉันสามารถข้าม `/plan` ได้หรือไม่?**
ตอบ: ได้ แต่ `/write` จะมีประสิทธิภาพมากขึ้นหากมี `/plan` เป็นแนวทาง
