# ปลั๊กอิน Authentic Voice (น้ำเสียงที่สมจริง)

ช่วยให้ตัวอักษรดูเหมือนเขียนขึ้นด้วยมือของคุณเอง: โดยเน้นย้ำไปที่รายละเอียดของชีวิต จังหวะของภาษาปาก และความสอดคล้องกันของคลังคำศัพท์เฉพาะบุคคล ปลั๊กอินนี้มุ่งเน้นไปที่การ "ยกระดับความเป็นต้นฉบับและความเป็นธรรมชาติ" เท่านั้น โดยไม่เกี่ยวข้องและไม่รองรับการใช้งานเพื่อหลบเลี่ยงการตรวจจับใดๆ ทั้งสิ้น

## การติดตั้ง


```

novel plugins add authentic-voice

```

## 的การเตรียมคลังข้อมูลภาษาเฉพาะบุคคล (Personal Corpus)

กรอกข้อมูลเพิ่มเติมในไฟล์ `.specify/memory/personal-voice.md` ดังนี้:
- คำติดปากและสำนวนที่ใช้บ่อย
- โครงสร้างประโยคประจำและจังหวะการเขียนที่ชอบ
- รายการประสบการณ์ชีวิต และคลังคำศัพท์เฉพาะทางในสายอาชีพ/ท้องถิ่น
- สิ่งที่หลีกเลี่ยงและข้อห้ามในการเขียน
- รสนิยมการเปรียบเปรยและคลังภาพสะท้อน (Imagery)

## คำสั่งที่ใช้บ่อย

- `/authentic-voice` เริ่มต้นระบบการเขียนด้วยน้ำเสียงที่สมจริง โดยระบบจะอ่านคลังข้อมูลภาษาเฉพาะบุคคลโดยอัตโนมัติ และจะแสดง "การ์ดรวบรวมวัตถุดิบ" (Material Card) ก่อนที่จะแสดงผลลัพธ์
- `/authenticity-audit` ทำการตรวจสอบความเป็นธรรมชาติ (ความเป็นมนุษย์) ของข้อความที่มีอยู่เดิม พร้อมทั้งให้คำแนะนำในการเรียบเรียงใหม่
- `/expert authentic-editor` อัญเชิญผู้เชี่ยวชาญด้านการบรรณาธิการน้ำเสียงสมจริง เพื่อทำการขัดเกลาและเรียบเรียงใหม่ในระดับบรรทัดอย่างละเอียดถี่ถ้วน

## ฉากทัศน์/สถานการณ์ที่เหมาะสม

- การสร้างสรรค์เนื้อหาฉบับจริง: เมื่อต้องการรักษาสไตล์และจังหวะที่เป็นเอกลักษณ์ส่วนตัวให้คงที่
- การขัดเกลาในรอบที่สอง: เพื่อหลีกเลี่ยงคำพูดที่ว่างเปล่า การเขียนตามเทมเพลตสำเร็จรูป หรือการใช้จังหวะการเล่าเรื่องที่ซ้ำซากจำเจ
- โปรเจกต์ระยะยาว: ค่อยๆ สะสมคลังคำศัพท์และภาพสะท้อนเฉพาะตัว เพื่อสร้าง "ลายนิ้วมือของผู้เขียน" (Author Fingerprint) ที่มั่นคงและชัดเจน
