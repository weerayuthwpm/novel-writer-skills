## เป้าหมาย
สร้างหลักการและค่านิยมหลักสำหรับการเขียนนิยาย เพื่อจัดทำเอกสาร "รัฐธรรมนูญ" แห่งการสร้างสรรค์ หลักการเหล่านี้จะเป็นแนวทางในการตัดสินใจสร้างสรรค์ทั้งหมดในขั้นตอนต่อไป

## ขั้นตอนการดำเนินการ
### 1. ตรวจสอบเอกสารที่มีอยู่
**ก่อนอื่นให้ตรวจสอบว่ามีเอกสารอ้างอิงสไตล์** (จาก `/book-internalize`) หรือไม่:
```bash
test -f memory/style-reference.md && echo "exists" || echo "not-found"
```
- หากมี ให้ใช้เครื่องมือ Read เพื่ออ่าน `memory/style-reference.md`
- จากนั้นแจ้งผู้ใช้ว่า: "ตรวจพบว่าคุณได้ทำการวิเคราะห์ผลงานอ้างอิงเรียบร้อยแล้ว ฉันจะอ้างอิงสไตล์ดังกล่าวเพื่อช่วยร่างรัฐธรรมนูญให้คุณ"

**จากนั้นตรวจสอบรัฐธรรมนูญที่มีอยู่**:
```bash
test -f memory/constitution.md && echo "exists" || echo "not-found"
```
- หากมี (แสดง "exists") ให้ใช้เครื่องมือ Read เพื่ออ่าน `memory/constitution.md` และเตรียมการอัปเดต
- หากไม่มี (แสดง "not-found") ให้ข้ามขั้นตอนการอ่านและเตรียมสร้างรัฐธรรมนูญใหม่ทันที

### 2. รวบรวมหลักการสร้างสรรค์
จากอินพุตของผู้ใช้ รวบรวมหลักการในมิติต่อไปนี้ (หากไม่ได้ให้มา ให้สอบถามหรืออนุมานเอา):

#### ค่านิยมหลัก
- ผลงานต้องการสื่อแนวคิดหลักอะไร?
- ขีดจำกัดขั้นต่ำที่ไม่สามารถละเมิดได้อย่างเด็ดขาดคืออะไร?
- จุดประสงค์พื้นฐานของการสร้างสรรค์คืออะไร?

#### มาตรฐานคุณภาพ
- ข้อกำหนดความสอดคล้องของตรรกะ
- มาตรฐานคุณภาพภาษา
- คำมั่นสัญญาความถี่ในการอัปเดต
- การรับประกันความสมบูรณ์

#### หลักการสไตล์การสร้างสรรค์
- สไตล์การเล่าเรื่อง (กระชับ/หรูหรา/เรียบง่าย/กวี)
- การควบคุมจังหวะ (รวดเร็ว/ช้า/ยืดหยุ่น)
- โทนอารมณ์ (ร้อนแรง/ลึกซึ้ง/สบายๆ/จริงจัง)
- เอกลักษณ์ภาษา (โบราณ/สมัยใหม่/ภาษาพูด/ลายลักษณ์อักษร)

#### หลักการเนื้อหา
- หลักการสร้างตัวละคร
    - ทุกตัวละครต้องมีแรงจูงใจที่สมบูรณ์
    - การเติบโตของตัวละครต้องสอดคล้องกับตรรกะ
    - บทสนทนาต้องสอดคล้องกับสถานะของตัวละคร
- หลักการออกแบบโครงเรื่อง
    - หลักการออกแบบความขัดแย้ง
    - ข้อกำหนดความสมเหตุสมผลของการหักมุม
    - หลักการการเก็บปม (Foreshadowing) ที่ปูไว้
- หลักการโลกทัศน์
    - ข้อกำหนดความสอดคล้องของกฎในโลก
    - มาตรฐานความสมจริงของรายละเอียด
    - ข้อกำหนดการตรวจสอบข้อมูลทางวัฒนธรรม

#### หลักการที่มุ่งเน้นผู้อ่าน
- การกำหนดกลุ่มผู้อ่านเป้าหมาย
- การรับประกันประสบการณ์ของผู้อ่าน
- หลักการตอบโต้และรับฟังความคิดเห็น

#### วินัยการสร้างสรรค์
- แนวปฏิบัติในการเขียนประจำวัน
- ขั้นตอนการแก้ไขและปรับปรุง
- หลักการจัดการเวอร์ชัน

4. การจัดการเวอร์ชัน
- เวอร์ชันหลัก: การเปลี่ยนแปลงหรือลบล้างหลักการครั้งสำคัญ
- เวอร์ชันรอง: การเพิ่มหลักการหรือบทใหม่
- หมายเลขแก้ไข: การปรับแก้ถ้อยคำ, การชี้แจง

5. การเผยแพร่ความสอดคล้อง
ตรวจสอบและอัปเดตเอกสารที่เกี่ยวข้องให้สอดคล้องกัน:
- อ้างอิงหลักการรัฐธรรมนูญในคำสั่งถัดไป
- แนะนำให้อัปเดตส่วนแนวคิดการสร้างสรรค์ใน README

6. สร้างรายงานผลกระทบ
ส่งออกผลกระทบของการสร้าง/อัปเดตรัฐธรรมนูญ:

  ✅ การจัดทำแผนต้องสอดคล้องกับหลักการ
  ✅ การดำเนินการสร้างสรรค์ต้องปฏิบัติตามข้อกำหนด

## หลักการปฏิบัติ
### ต้องปฏิบัติตาม
- หลักการต้องสามารถตรวจสอบได้ ไม่เป็นนามธรรมจนเกินไป
- ใช้คำที่ชัดเจนเช่น "ต้อง", "ห้าม"
- ทุกหลักการต้องมีเหตุผลที่ชัดเจน

### ควรมี
- ค่านิยมหลักอย่างน้อย 3-5 ข้อ
- มาตรฐานคุณภาพขั้นต่ำที่ชัดเจน
- ข้อกำหนดการสร้างสรรค์ที่สามารถปฏิบัติได้จริง

### ควรหลีกเลี่ยง
- สโลแกนที่กว้างเกินไป (เช่น "มุ่งมั่นสู่ความเป็นเลิศ")
- ข้อกำหนดที่ไม่สามารถตรวจสอบได้
- ข้อบังคับที่จำกัดความคิดสร้างสรรค์มากเกินไป

## ตัวอย่างหลักการ
**หลักการที่ดี**:
- "พฤติกรรมของตัวละครหลักต้องมีห่วงโซ่แรงจูงใจที่ชัดเจน ห้ามปรากฏพฤติกรรมที่ทำเพียง 'เพราะความจำเป็นของเนื้อเรื่อง'"
- "การปูปมทุกจุดต้องได้รับการเก็บกู้หรืออธิบายภายในเวลาที่สมเหตุสมผล (ไม่เกิน 10 บท)"
- "ห้ามใช้ศัพท์สแลงอินเทอร์เน็ตสมัยใหม่ที่ทำลายความรู้สึกดื่มด่ำของฉากหลังยุคโบราณอย่างเด็ดขาด"

**หลักการที่แย่**:
- "ต้องเขียนให้ดีๆ" (กว้างเกินไป)
- "มุ่งเน้นความเป็นศิลปะ" (ไม่สามารถตรวจสอบได้)
- "ทำให้ผู้อ่านพึงพอใจ" (มาตรฐานไม่ชัดเจน)

## ขั้นตอนต่อไป
หลังจากกำหนดรัฐธรรมนูญแล้ว ขั้นตอนการสร้างสรรค์ทั้งหมดในลำดับต่อไปต้องปฏิบัติตาม:
1.`/specify` - สเปกต้องสอดคล้องกับค่านิยมของรัฐธรรมนูญ
2.`/plan` - แผนต้องปฏิบัติตามหลักการของรัฐธรรมนูญ
3.`/write` - การสร้างสรรค์ต้องปฏิบัติตามข้อกำหนดของรัฐธรรมนูญ
4.`/analyze` - การตรวจสอบต้องตรวจสอบความสอดคล้องกับรัฐธรรมนูญ
