# การจัดการความสัมพันธ์ของตัวละคร

ติดตามและจัดการพลวัตของความสัมพันธ์ระหว่างตัวละคร พร้อมรับประกันความสมเหตุสมผลของการพัฒนาความสัมพันธ์

## ฟังก์ชัน

1. **เครือข่ายความสัมพันธ์** - ดูแลแผนผังความสัมพันธ์ระหว่างตัวละคร
2. **การเปลี่ยนแปลงความสัมพันธ์** - บันทึกประวัติวิวัฒนาการของความสัมพันธ์
3. **การจัดการฝ่าย/กลุ่มก้อน** - ติดตามการต่อสู้และการร่วมมือกันระหว่างกลุ่มอำนาจต่างๆ
4. **การติดตามอารมณ์** - จัดการพัฒนาทางอารมณ์ระหว่างตัวละคร

## วิธีการใช้งาน

รันสคริปต์ {SCRIPT} [ปฏิบัติการ] [พารามิเตอร์]：
- `update` - อัปเดตความสัมพันธ์ของตัวละคร
- `show` - แสดงเครือข่ายความสัมพันธ์
- `history` - ดูประวัติการเปลี่ยนแปลงความสัมพันธ์
- `check` - ตรวจสอบตรรกะความสัมพันธ์

ตัวอย่าง：
```
{SCRIPT} update หลี่จงยง allies เฉินอวี่ชิง --chapter 61 --note ช่วยเหลือตอนเข้ารับราชการในฮั่นหลิน
# PowerShell:
{SCRIPT} -Command update -A หลี่จงยง -Relation allies -B เฉินอวี่ชิง -Chapter 61 -Note ช่วยเหลือตอนเข้ารับราชการในฮั่นหลิน
```

  }

## ตัวอย่างผลลัพธ์

```
👥 เครือข่ายความสัมพันธ์ของตัวละคร
━━━━━━━━━━━━━━━━━━━━
ตัวเอก：หลี่จงยง
├─ 💕 หลงรัก：เฉินอวี่ชิง
├─ 🤝 พันธมิตร：จางจูเจิ้ง (แอบแฝง)
├─ 📚 อาจารย์：มัตเตโอ ริชชี
├─ ⚔️ ศัตรู：กลุ่มของเสินซื่อซิง
└─ 👁️ สอดส่อง：ตงฉาง (สำนักจารชน)

การต่อสู้ระหว่างฝ่าย：
ฝ่ายปฏิรูป ←→ ฝ่ายอนุรักษ์นิยม
ตงหลิน ←→ กลุ่มขันที

การเปลี่ยนแปลงล่าสุด (ตอนที่ 60)：
- เฉินอวี่ชิง：คนแปลกหน้า → ต่างฝ่ายต่างถูกใจ
- จางจูเจิ้ง：ไม่ทราบ → ความสัมพันธ์ศิษย์-อาจารย์
```
