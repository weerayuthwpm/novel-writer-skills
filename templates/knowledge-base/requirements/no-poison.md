# ข้อกำหนดการเขียนแนวไร้พิษ (No-Poison / พล็อตไม่ออกทะเล)

## แนวคิดหลัก

**ค่านิยมถูกต้อง ตรรกะสมเหตุสมผล** — ทำให้ผู้อ่านรู้สึกสบายใจ ไม่รู้สึกขัดใจหรืออึดอัด

---

## รายการ "จุดพิษ" ที่พบบ่อย

### ❌ สิ่งที่ต้องหลีกเลี่ยงโดยเด็ดขาด

#### 1. ตัวเอกไอคิวลด

```

❌ ทั้งที่รู้ว่าเป็นกับดักแต่ก็ยังกระโดดใส่
❌ ลืมใช้ทักษะ/สกิลที่มีในสถานการณ์คับขัน
❌ มุทะลุ ไร้สมอง

```

#### 2. การบังคับให้เข้าใจผิดกัน

```

❌ เรื่องที่พูดคำเดียวก็เข้าใจกันได้ แต่กลับไม่ยอมพูด
❌ แอบฟังคนอื่นคุยกันแต่ฟังแค่ครึ่งๆ กลางๆ
❌ ไม่ยอมเปิดโอกาสให้อีกฝ่ายได้อธิบาย

```

#### 3. พล็อตประเภทพ่อพระ/แม่พระเกินพอดี

```

❌ ให้อภัยคนอื่นอย่างไม่มีขอบเขตและไม่มีจุดยืน
❌ ฟอกขาวให้ศัตรู/ตัวร้ายกลับตัวเป็นคนดีแบบไม่มีเหตุผล
❌ เมตตาต่อศัตรู แต่กลับโหดร้ายใจดำกับพวกพ้องตัวเอง

```

#### 4. ตัวละครสมทบทำหน้าที่เป็นแค่เครื่องมือ (Tool)

```

❌ มีตัวตนอยู่เพียงเพื่อมาแจกแต้ม/หาที่ตายเท่านั้น
❌ มีตัวตนอยู่เพียงเพื่อเป็นฉากหลังประกอบ
❌ ไม่มีแรงจูงใจในการกระทำที่สมเหตุสมผล

```

#### 5. พล็อตสองมาตรฐาน (Double Standard)

```

❌ ตัวเอกทำได้ แต่คนอื่นห้ามทำ
❌ ทีเจ้าเมืองจุดไฟได้ (แต่ชาวบ้านห้ามจุดตะเกียง) / ตนเองทำผิดไม่เป็นไร คนอื่นทำผิดไม่ได้

```

## รายการตรวจสอบ (Checklist)

ตรวจสอบตัวเองหลังจากเขียนเสร็จในแต่ละบท:
- [ ] พฤติกรรมและการกระทำของตัวเอกสมเหตุสมผลหรือไม่?
- [ ] ตัวละครสมทบมีแรงจูงใจในการกระทำที่สมเหตุสมผลหรือยัง?
- [ ] ความเข้าใจผิดในเรื่องสามารถหลีกเลี่ยงได้หรือไม่?
- [ ] ปมขัดแย้งเกิดขึ้นอย่างเป็นธรรมชาติหรือไม่?
