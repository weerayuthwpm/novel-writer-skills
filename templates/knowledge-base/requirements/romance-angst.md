# ข้อกำหนดการเขียนแนวนิยายรักสายดราม่า/ตับพัง (Romance-Angst)

## แนวคิดหลัก

**ดราม่าอย่างมีเหตุผล ดราม่าอย่างมีขอบเขต** — ทำให้ผู้อ่านเจ็บปวดสะใจ ไม่ใช่ดราม่าจนรู้สึกรำคาญ

---

## ข้อกำหนดสำคัญ

### 1. เหตุผลของการดราม่า

**ต้องมีเหตุผลรองรับที่หนักแน่นพอ**:
- ความเข้าใจผิด (แต่ต้องสมเหตุสมผล)
- อุปสรรคจากภายนอก (ครอบครัว/ตระกูล, โชคชะตา)
- ข้อบกพร่องทางปมขัดแย้งในนิสัยของตัวละคร
- บาดแผลในอดีต (Trauma)

**ห้ามสร้างสถานการณ์ดราม่าขึ้นมาลอยๆ โดยไม่มีปี่มีขลุ่ย**

### 2. กระบวนการดำเนินความดราม่า

**กระบวนการมาตรฐาน**:

```

ช่วงหวานชื่น (สร้างความผูกพันทางอารมณ์)
↓
วิกฤตการณ์เริ่มปรากฏ
↓
การพลัดพราก / ความเข้าใจผิด
↓
ต่างฝ่ายต่างจมอยู่กับความทุกข์ระทม
↓
ความจริงเปิดเผย
↓
การปรับความเข้าใจ (หรือจบแบบ Bad Ending)

```

### 3. ขอบเขตของความดราม่า

**ความดราม่าที่อนุญาตให้มีได้**:
- การทรมานจิตใจ/ความบอบช้ำทางจิตใจ
- ความทุกข์ทรมานจากความคัดคะนึงหา
- ความเจ็บปวดจากการที่ไม่สามารถครองคู่กันได้

**ความดราม่าที่ต้องใช้อย่างระมัดระวัง**:
- การทำร้ายร่างกาย
- การหักหลัง/ทรยศ (จำเป็นต้องมีคำอธิบายที่สมเหตุสมผล)
- มือที่สาม (แต่ต้องไม่ได้รักหรือลงเอยกันจริงๆ)

**ความดราม่าที่ห้ามมีโดยเด็ดขาด**:
- การทารุณกรรมอย่างไร้เหตุผล
- การหักหลังอย่างไม่มีจุดสิ้นสุดและไม่มีขอบเขต
- การจงใจยัดความรุนแรงใส่พล็อตเพื่อดราม่าอย่างไร้สาระ (ดราม่าเพื่อดราม่า)

### 4. การคลี่คลายหลังผ่านพ้นความดราม่า

**ในกรณีที่จบแบบ Happy Ending (HE)**:
- ต้องมีกระบวนการปรับความเข้าใจและคืนดีกัน
- อยู่ๆ จะกลับมาดีกันทันทีโดยไม่มีปี่มีขลุ่ยไม่ได้
- จำเป็นต้องมีการขอโทษ การอธิบาย และการให้อภัย

**ในกรณีที่จบแบบ Bad Ending (BE)**:
- ต้องให้บทสรุปและคำอธิบายที่เคลียร์แก่ผู้อ่าน
- ห้ามตัดจบแบบเผางานหรือออกทะเลเด็ดขาด

## การออกแบบจุดดราม่าเรียกน้ำตา

### รูปแบบที่ 1: ดราม่าจากความเข้าใจผิด

```

เธอเข้าใจผิดคิดว่าเขาไม่ได้รักเธออีกต่อไปแล้ว
เขาเข้าใจผิดคิดว่าเธอเปลี่ยนใจไปรักคนอื่น
ทั้งสองต่างจมอยู่กับความเจ็บปวด แต่กลับไม่มีใครยอมเปิดปากพูดความจริง

```

### รูปแบบที่ 2: ดราม่าจากการพลัดพราก

```

ทั้งที่รักกันลึกซึ้ง แต่กลับไม่สามารถอยู่ร่วมกันได้
ระยะทาง ตัวตนฐานะ หรือโชคชะตา ล้วนกลายมาเป็นอุปสรรคขวางกั้น

```

### รูปแบบที่ 3: ดราม่าจากจังหวะเวลา (โชคชะตาเล่นตลก)

```

ในยามที่เขารักเธอ เธอ กลับไม่ได้รักเขา
ในยามที่เธอรักเขา เขา กลับมีคนอื่นเคียงข้างไปเสียแล้ว
การคลาดกันในจังหวะเวลา คือความเจ็บปวดที่กรีดลึกที่สุด

```

## ระดับความดราม่าปวดตับ

- ระดับเริ่มต้น: ความขัดแย้งเล็กๆ น้อยๆ
- ระดับปานกลาง: รักๆ เลิกๆ แยกจากแล้วกลับมาเจอกัน
- ระดับรุนแรง: พลัดพรากจากกันชั่วนิรันดร์ / ความเป็นความตาย
- ระดับขั้นสุด: ตอนจบแบบตับพังพินาศ / Bad Ending (BE)
