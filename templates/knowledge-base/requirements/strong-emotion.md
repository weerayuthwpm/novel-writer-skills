# ข้อกำหนดการเขียนแนวอารมณ์เข้มข้น (Strong-Emotion)

## แนวคิดหลัก

**อารมณ์อัดแน่นเต็มพิกัด แรงกระแทกกระทั้นสูง** — ทำให้ผู้อ่านได้รับประสบการณ์ทางอารมณ์ที่รุนแรงสะเทือนใจ

---

## ข้อกำหนดสำคัญ

### 1. จุดระเบิดทางอารมณ์ (Emotion Eruption Points)

ทุกๆ 10-15 บท ต้องมีจุดระเบิดทางอารมณ์อย่างน้อย 1 จุด:
- การระเบิดความโกรธแค้น
- ความโศกเศร้าที่ทลายทำนบ
- ความปิติยินดีที่เปี่ยมล้น
- ความหวาดกลัวถึงขีดสุด

### 2. การปูอารมณ์ (Emotional Pacing)

**อยู่ๆ จะระเบิดอารมณ์ขึ้นมาทันทีไม่ได้ ต้องมีการปูเนื้อเรื่องล่วงหน้า**:

```

กดดัน/อึดอัด (3 บท) → สะสมความรู้สึก (2 บท) → ระเบิดอารมณ์ (1 บท)

```

### 3. การแสดงออกทางอารมณ์

#### ✅ แสดงออกผ่านพฤติกรรม

```

ความโกรธ: ขว้างปาข้าวของ, ตัวสั่นเทิ้ม, กัดฟันแน่น
ความเศร้า: ร้องไห้ไร้เสียง, คู้ตัวกอดตัวเอง, นั่งเหม่อลอย
ความสุข: หัวเราะลั่น, กระโดดโลดเต้น, โผเข้ากอด

```

#### ❌ บรรยายบอกตรงๆ

```

"เธอโกรธมาก" → ยังดีไม่พอ
"เธอโกรธจนตัวสั่นไปหมด" → ดีขึ้น
"เธอคว้าถ้วยขว้างใส่กำแพง" → ดีที่สุด

```

## จังหวะทางอารมณ์ (Emotional Rhythm)

**ดำเนินเรื่องเป็นลูกคลื่น**:

```

จุดไคลแมกซ์ → ช่วงผ่อนคลาย → จุดไคลแมกซ์ → ช่วงผ่อนคลาย

```

**สิ่งที่ห้ามทำ**:

```

พีคสูงตลอดเวลา (ผู้อ่านจะเกิดอาการชาชิน)
เรื่อยๆ มาเรียงๆ ตลอดเวลา (ผู้อ่านจะเบื่อหน่าย)

```
