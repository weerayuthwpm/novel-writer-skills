# คลังความรู้สไตล์การเขียน (Writing Style Knowledge Base)

สารบัญนี้ประกอบด้วยสไตล์การเขียนสำเร็จรูป 5 รูปแบบ ซึ่งจะถูกตรวจจับและโหลดโดยอัตโนมัติผ่านสกิล **style-detector**

---

## 🎨 สไตล์การเขียนที่พร้อมใช้งาน

### 1. natural-voice (สไตล์เสียงธรรมชาติ/ภาษาพูด)

**บริบทที่เหมาะสม**: นวนิยายรัก, แนวเมืองยุคปัจจุบัน, แนวสมจริง (Realistic Fiction)

**คุณลักษณะเด่น**:
- เน้นการใช้ภาษาพูดเป็นหลัก ให้ความรู้สึกเหมือนกำลังเล่าเรื่องให้เพื่อนฟัง
- รายละเอียดชีวิตประจำวันเหลือล้น สร้างความรู้สึกร่วม (Immersive) ได้ง่าย
- ใช้บทสนทนาในการขับเคลื่อนเนื้อเรื่อง (คิดเป็นสัดส่วน 40-60%)
- หลีกเลี่ยงภาษาเขียนที่มีสำนวนประดิษฐ์ประดอยหรือภาษาจรรโลงใจแบบวรรณกรรม

**คำสำคัญในการเปิดใช้งาน**:

```

ภาษาพูด, อิงชีวิตจริง, สมจริง, ติดดิน/เข้าถึงง่าย
เหมือนคนคุยกัน, อย่าใช้ภาษาประดิษฐ์, ตัดภาษาเขียนออก
เป็นธรรมชาติ, ไม่เก๊ก, ภาษาชาวบ้าน, บทสนทนาในชีวิตประจำวัน

```

**ตัวอย่างเปรียบเทียบ**:

```

❌ กลิ่นอาย AI แรง:
หลี่หว่านฉิงนั่งลงที่มุมหนึ่งของร้านกาแฟ นิ้วมือเรียวยาวลูบไล้ไปบนผนังถ้วยอย่างแผ่วเบา
สัมผัสอันอบอุ่นทำให้ความคิดของเธอค่อยๆ สงบลง

✅ natural-voice:
หลี่หว่านฉิงหาที่นั่งริมหน้าต่าง กาแฟค่อนข้างร้อน เธอเลยเปลี่ยนมือถือ
แดดข้างนอกค่อนข้างแสบตา เธอเลยหรี่ตาลงนิดหน่อย

```

### 2. literary (สไตล์วรรณศิลป์)

**บริบทที่เหมาะสม**: วรรณกรรมจริงจัง, วรรณกรรมบริสุทธิ์, นิยายอิงประวัติศาสตร์ (ที่มีฉากหลังร่วมสมัย)

**คุณลักษณะเด่น**:
- มีความงดงาม ลึกซึ้ง และมีคุณค่าทางวรรณศิลป์
- มุ่งเน้นการสร้างบรรยากาศเชิงอารมณ์ และแง่คิดเชิงปรัชญา
- มีบทบรรยายความรู้สึกนึกคิดในใจที่พอเหมาะ
- ภาษาที่ใช้มีความยับยั้งชั่งใจแต่ทรงพลัง

**คำสำคัญในการเปิดใช้งาน**:

```

มีศิลปะทางวรรณกรรม, งดงาม, ลึกซึ้ง, มีมิติ
วรรณกรรมจริงจัง, วรรณกรรมบริสุทธิ์, บรรยากาศเชิงอารมณ์, บทกวี (ร้อยแก้วร่วมสมัย)
ปรัชญา, ความเป็นมนุษย์, มีแง่คิดเชิงอุดมการณ์

```

**ตัวอย่างเปรียบเทียบ**:

```

❌ นิยายออนไลน์ทั่วไป:
เธอเสียใจมาก ไม่รู้จะทำยังไงดี ข้างนอกฝนตกอยู่

✅ literary:
กลิ่นหอมของกาแฟอวลกระจายในอากาศอย่างช้าๆ หลี่หว่านฉิงนั่งอยู่ตรงตำแหน่งริมหน้าต่าง
ถ้วยในมือค่อยๆ เย็นชืดลง ราวกับอารมณ์ของเธอในยามนี้ แสงแดดยามบ่ายถูกตัดทอนเป็นเส้นสาย
เธอนึกถึงช่วงเวลาหนึ่งเมื่อหลายปีก่อน — แสงแดดแบบเดียวกัน การรอคอยแบบเดียวกัน
ทว่าในตอนนั้น เธอยังคงเชื่อว่าการรอคอยนั้นมีความหมาย

```

### 3. web-novel (สไตล์นิยายออนไลน์ยอดนิยม)

**บริบทที่เหมาะสม**: แฟนตาซี, เซียนเซี่ย/เทพเซียน, นิยายเมืองแนวสะใจ, แนวระบบ (System)

**คุณลักษณะเด่น**:
- เดินเรื่องไว ปมขัดแย้งหนาแน่น
- จุดสะใจเกิดขึ้นบ่อย (ตบหน้า, โชว์เทพ, อัปเลเวล)
- บทสนทนาสั้นกระชับและทรงพลัง
- ใช้พล็อตสูตรสำเร็จอย่างชัดเจนแต่ไม่ซ้ำซากจำเจ

**คำสำคัญในการเปิดใช้งาน**:

```

นิยายออนไลน์, นิยายเน้นความสะใจ, ดำเนินเรื่องเร็ว, พล็อตสูตรสำเร็จ
ตบหน้า, โชว์เทพ, แนวอัปเลเวล, แนวพระเอกขยะตอนแรก
สไตล์แฟนตาซี, สไตล์เซียนเซี่ย, นิยายแนวระบบ

```

**ตัวอย่างเปรียบเทียบ**:

```

❌ ยืดเยื้อ:
หลี่หมิงค่อยๆ ก้าวเดินขึ้นไปบนลานประลอง ในใจมีความคิดร้อยแปดประดังเข้ามา
พาลนึกย้อนไปถึงเส้นทางการฝึกตนตลอดหลายปีที่ผ่านมา...

✅ web-novel:
"หลี่หมิง?" นายน้อยจางแค่นยิ้มอย่างเย็นชา "แกคือไอ้ขยะที่โดนถอนหมั้นคนนั้นน่ะเหรอ?"

สิ้นเสียงคำพูด ประกายกระบี่สายหนึ่งก็พลันพาดผ่าน

รอยยิ้มของนายน้อยจางแข็งค้างอยู่บนใบหน้า วินาทีต่อมา แขนเสื้อของเขาก็ร่วงหล่นลงพื้นอย่างไร้เสียง

"ขออภัย" หลี่หมิงเก็บกระบี่ "มือลื่นไปหน่อย"

ทั่วทั้งลานประลองเงียบกริบราวป่าช้า

```

### 4. ancient-style (สไตล์กลิ่นอายโบราณ)

**บริบทที่เหมาะสม**: กำลังภายใน, เซียนเซี่ย/เทพเซียน (ในแง่ของสำนวนภาษา), นิยายที่มีฉากหลังเป็นยุคโบราณ

**คุณลักษณะเด่น**:
- มีกลิ่นอายโบราณที่งดงาม แต่ไม่ได้ใช้ภาษาโบราณ จนอ่านยากเกินไป
- อุดมไปด้วยสัญลักษณ์เปรียบเปรย ศิลปะแห่งการเว้นช่องว่างให้คิด
- บทบรรยายมีท่วงทำนองราวกับบทกวี
- หลีกเลี่ยงการใช้คำศัพท์สมัยใหม่

**คำสำคัญในการเปิดใช้งาน**:

```

ภาษากลุ่มโบราณ, กลิ่นอายโบราณ, ภาษาเขียนโบราณ (ไม่เอาหนาเกินไป)
ความงดงามเชิงกวี, ภาพลักษณ์เปรียบเปรย, ศิลปะการเว้นช่องว่าง, แนวท่องยุทธภพ
ความงามแบบคลาสสิก, ขนบธรรมเนียมดั้งเดิม, กลิ่นอายชาวชาวยุทธ

```

**ตัวอย่างเปรียบเทียบ**:

```

❌ มีความสมัยใหม่เกินไป:
หลี่หว่านฉิงโกรธมาก เธอรู้สึกว่าตัวเองถูกหลอกลวง

✅ ancient-style:
หว่านฉิงถือกระบี่หยัดยืน สายลมพัดโบกกระพือชายเสื้อของเธอให้ปลิวไสว
"ท่านลวงข้า"
คำพูดสี่พยางค์ แผ่วเบาราวกับเสียงทอดถอนใจ ทว่ากลับกรีดลึกราวกับใบมีดในทุกถ้อยคำ

```

### 5. minimal (สไตล์มินิมอล/สั้นกระชับ)

**บริบทที่เหมาะสม**: ระทึกขวัญ, สืบสวนสอบสวน, แนวสืบสวนด้านมืด (Hard-boiled), นิยายแนวทดลอง (Experimental)

**คุณลักษณะเด่น**:
- กระชับขั้นสุด ตัดส่วนขยายทิ้งทั้งหมด
- เน้นใช้ประโยคสั้น จังหวะเรื่องหนักแน่นดิบเถื่อน
- ใช้การเว้นช่องว่างให้คิด ไม่มีการอธิบายขยายความ
- มีความยับยั้งชั่งใจในสไตล์การเขียนของเฮมิงเวย์ (Hemingway Style)

**คำสำคัญในการเปิดใช้งาน**:

```

มินิมอล, กระชับ, ยับยั้งชั่งใจ, สไตล์เฮมิงเวย์
ดิบเถื่อน/ดุดัน, พูดเนื้อๆ ไม่เอาน้ำ, ไม่ลนลาน, เว้นให้คิด
ไม่ต้องอธิบาย, สไตล์ลูกผู้ชายตัวจริง

```

**ตัวอย่างเปรียบเทียบ**:

```

❌ ฟุ่มเฟือย:
หลี่หว่านฉิงค่อยๆ ผลักประตูเปิดออก แล้วพบว่าภายในห้องนั้นมืดมิดไปหมด
เธอเอ่ยถามขึ้นมาด้วยความรู้สึกฉงนใจอยู่บ้าง...

✅ minimal:
หลี่หว่านฉิงผลักประตู มืดสนิท

"ทำไมไม่เปิดไฟ?"
"ลืม"
"ป่วยเหรอ?"

เธอเอามือจับหน้าผากเขา ร้อนจี๋

```

## 🔍 กลไกการทำงาน

### ขั้นตอนการตรวจจับอัตโนมัติ

1. **ดักฟังข้อมูลนำเข้าของผู้ใช้**
   - ตรวจสอบคำสำคัญภายในคำสั่ง `/specify`, `/constitution` และบทสนทนา
   - ระบุคำสำคัญของสไตล์การเขียน

2. **เปิดใช้งาน style-detector**
   - วิเคราะห์คำสำคัญ
   - จับคู่กับสไตล์การเขียนที่ตรงกัน

3. **โหลดคลังความรู้**
   - อ่านไฟล์สไตล์การเขียนที่เกี่ยวข้อง
   - แสดงประเด็นสำคัญหลัก

4. **บันทึกการตั้งค่า**
   - ทำเครื่องหมายระบุสไตล์ไว้ใน `specification.md`
   - นำไปใช้โดยอัตโนมัติเมื่อมีการเรียกคำสั่ง `/write`

### การจัดการกรณีสไตล์ผสม (Multi-Style)

หากตรวจพบคำสำคัญของสไตล์การเขียนมากกว่าหนึ่งสไตล์ ระบบจะแจ้งให้ผู้ใช้เลือกสไตล์หลักหรือกำหนดสัดส่วนการผสม:


```

ตรวจพบ: natural-voice + ancient-style

แนวทางที่แนะนำ:

1. ใช้สไตล์หลักเป็น natural-voice: บทสนทนายังคงใช้ภาษาพูด แต่ในส่วนบทบรรยายจะสอดแทรกกลิ่นอายและภาพลักษณ์โบราณคลาสสิกเข้าไป
2. ใช้สไตล์หลักเป็น ancient-style: เนื้อหาแกนหลักเน้นความโบราณคลาสสิก แต่ปรับบทสนทนาให้มีความเป็นภาษาพูดที่เข้าใจง่ายขึ้นเมื่อเหมาะสม

```

## 📚 การทำงานร่วมกับระบบอื่น

### ความสัมพันธ์กับคลังความรู้ประเภทนิยาย (genres/)

- **คลังความรู้ประเภทนิยาย**: มอบโครงสร้างเรื่องและพล็อตสำเร็จรูป (เช่น พัฒนาการความสัมพันธ์ในนิยายรัก, การจัดวางเบาะแสในนิยายสืบสวน)
- **คลังความรู้สไตล์การเขียน**: กำหนด "เสียง (Voice)" และ "รสชาติ (Tone)" ของตัวอักษร

สามารถนำมาผสมผสานใช้งานร่วมกันได้ เช่น:
- genres/romance.md (พล็อตนิยายรัก) + styles/natural-voice.md (สไตล์ภาษาพูด) = นิยายรักร่วมสมัยที่ใช้ภาษาพูดเข้าถึงง่าย
- genres/mystery.md (โครงสร้างสืบสวน) + styles/minimal.md (สไตล์มินิมอล) = นิยายสืบสวนสอบสวนแนว Hard-boiled ดิบเถื่อนกระชับ

### ความสัมพันธ์กับข้อกำหนดการเขียน (requirements/)

- **สไตล์การเขียน**: ตัวอักษรจะถูกเขียนออกมาอย่างไร (ในระดับสำนวนภาษา)
- **ข้อกำหนดการเขียน**: ต้องผ่านเกณฑ์มาตรฐานอะไรบ้าง (ในระดับกฎเกณฑ์ข้อบังคับ)

สามารถนำมาผสมผสานใช้งานร่วมกันได้ เช่น:
- natural-voice + anti-ai-v4 = ใช้สไตล์ภาษาพูดที่เป็นธรรมชาติ พร้อมทั้งล้างกลิ่นอาย AI ออกอย่างทรงพลัง
- web-novel + fast-paced = ได้สำนวนภาษาแบบนิยายออนไลน์ยอดนิยม พร้อมจังหวะเรื่องที่รวดเร็วฉับไว

## ⚙️ ตัวอย่างการตั้งค่า

### การระบุเครื่องหมายในไฟล์ specification.md

```yaml
---
writing-style: natural-voice
style-intensity: 80%  # ทางเลือกเพิ่มเติม: ระดับความเข้มข้นของสไตล์
---
```

### การตั้งค่าสไตล์แบบผสม (Hybrid Style Configuration)

```yaml
---
writing-style:
  primary: natural-voice (70%)
  secondary: literary (30%)
---
```

## 📝 คำแนะนำในการใช้งาน

### สำหรับนักเขียนมือใหม่
แนะนำสไตล์: **natural-voice** หรือ **web-novel**
- เริ่มต้นใช้งานได้ง่ายที่สุด
- เป็นที่ยอมรับและเข้าถึงกลุ่มผู้อ่านได้กว้างขวาง
- เกิดข้อผิดพลาดในพล็อตและภาษาได้ยาก

### สำหรับนักเขียนที่มีประสบการณ์
สามารถทดลองสไตล์: **literary** หรือ **minimal**
- สร้างเอกลักษณ์เฉพาะบุคคลได้โดดเด่นยิ่งขึ้น
- จำเป็นต้องมีความสามารถในการควบคุมทิศทางภาษาที่แข็งแกร่ง
- เหมาะกับวัตถุดิบและเนื้อหาเฉพาะทาง

### การเลือกตามเนื้อเรื่องเฉพาะแนว
- กำลังภายใน / เทพเซียน → **ancient-style**
- สืบสวนสอบสวน / ระทึกขวัญ → **minimal**
- นิยายเมืองยุคปัจจุบัน → **natural-voice**
- วรรณกรรมจริงจัง → **literary**
- นิยายออนไลน์แนวแฟนตาซีสะใจ → **web-novel**

## 🆕 การกำหนดสไตล์ด้วยตนเอง (Custom Style)

หากสไตล์สำเร็จรูปที่ระบบเตรียมไว้ให้ยังไม่ตอบโจทย์ คุณสามารถทำตามขั้นตอนต่อไปนี้ได้:

1. คัดลอกไฟล์สไตล์ที่มีความใกล้เคียงกับความต้องการของคุณมากที่สุด
2. ปรับแต่งเนื้อหาแก้ไขตามความต้องการ
3. บันทึกและตั้งชื่อสไตล์การเขียนใหม่
4. ระบุระบุชื่อสไตล์นั้นลงในไฟล์ `specification.md`

```yaml
---
writing-style: custom/my-style
---
```

---

## 📖 เอกสารโดยละเอียด

สามารถดูข้อกำหนดและรายละเอียดฉบับเต็มของแต่ละสไตล์ได้จากไฟล์ `.md` ที่เกี่ยวข้อง:
- `natural-voice.md` — คู่มือสไตล์เสียงธรรมชาติ/ภาษาพูดฉบับสมบูรณ์
- `literary.md` — คู่มือสไตล์วรรณศิลป์ฉบับสมบูรณ์
- `web-novel.md` — คู่มือสไตล์นิยายออนไลน์ยอดนิยมฉบับสมบูรณ์
- `ancient-style.md` — คู่มือสไตล์กลิ่นอายโบราณฉบับสมบูรณ์
- `minimal.md` — คู่มือสไตล์มินิมอลสั้นกระชับฉบับสมบูรณ์
