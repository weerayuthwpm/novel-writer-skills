# สไตล์การเขียนแนวโบราณ/กำลังภายใน (Ancient-Style)

## การกำหนดสไตล์

**แนวคิดหลัก**: กลิ่นอายโบราณอารมณ์ร่วมปัจจุบัน เน้นอารมณ์ความรู้สึกและภาพจำลองมาเป็นอันดับแรก

มีคติความงามแบบคลาสสิกแต่ไม่ใช่ภาษาโบราณจ๋า มีความกวีแต่ไม่คลุมเครือเข้าใจยาก

---

## บริบทที่เหมาะสม

### ✅ เหมาะที่สุด

* นิยายกำลังภายใน


* นิยายแนวเซียน/เทพเซียน (ในแง่ของสำนวนภาษา)


* แนวรักโรแมนติกฉากหลังยุคโบราณ


* แนวแฟนตาซีโบราณ



### ⚠️ ควรใช้อย่างระมัดระวัง

* เมืองกรุงยุคปัจจุบัน (จะดูขัดตามาก)


* แนวไซไฟล้ำยุค



---

## หลักการสำคัญ

### 1. ภาษาปัจจุบันเป็นหลัก แต่งแต้มด้วยกลิ่นอายโบราณ

**ไม่ใช่ภาษาโบราณคร่ำครึ แต่เป็นภาษาปัจจุบันที่มีกลิ่นอายย้อนยุค**

#### ❌ ภาษาโบราณแท้ (เข้าใจยากเกินไป)

```
ยามสายัณห์หว่านฉิงจับกระบี่ยืนหยัด ลมพัดอาภรณ์พริ้วไหว แววตาคมกล้าจับจ้องเบื้องหน้า
ศัตรูเยื้องกราย แสงกระบี่พลันวาบ ดุจมังกรทะยานจากห้วงอเวจี ยากจักต้านทาน

```

#### ✅ ภาษาปัจจุบันกลิ่นอายโบราณ

```
หว่านฉิงยืนอยู่ตรงนั้น ในมือซอนกุมกระบี่ไว้
สายลมพัดโบกโบยจนชายเสื้อของนางพริ้วไหว
นางมองคนตรงหน้า นิ่งเงียบไม่เอ่ยคำ

กระบี่ ออกจากฝักแล้ว

ประกายแสงสายหนึ่งวาบผ่าน รวดเร็วราวสายฟ้าแลบ

```

**จุดสำคัญ**:

* เลือกใช้คำโบราณที่คุ้นหู (ชายเสื้อ, ออกจากฝัก)


* รูปประโยคแบบปัจจุบัน (นาง, ไม่เอ่ยคำ)


* ผสมผสานใช้อย่างลงตัว



---

### 2. ภาพจำลองต้องแจ่มชัด

**ใช้สิ่งของหรือธรรมชาติมาแทนการบรรยายความรู้สึกตรงๆ**

#### ❌ บรรยายตรงๆ

```
นางโศกเศร้ามาก ร้องไห้อยู่คนเดียวเป็นเวลานาน

```

#### ✅ บรรยายด้วยภาพจำลอง

```
นางนั่งเหม่อลอยอยู่ริมหน้าต่างเพียงลำพัง
เปลวเทียนสั่นไหว ทอดเงาวูบวาบบนผนัง
หยาดน้ำตา หยดลงบนสายพิณ

ตัวพิณ นิ่งเงียบไร้เสียง

```

**ภาพจำลองที่นิยมใช้**:

* ลม, ฝน, หิมะ, พระจันทร์


* กระบี่, พิณ, หมากรุก, ตำรา


* ดอกไม้, ใบไม้, เมฆ, ควัน



---

### 3. ศิลปะแห่งการเว้นว่างและความเป็นกวี

**แก่นแท้ของแนวโบราณคือการเว้นช่องว่างให้คิด (Leave Blank)**

#### ✅ ตัวอย่าง

```
เขาจากไปแล้ว

สายลมยังคงพัดโบก
ใบไม้ยังคงร่วงหล่น

นางยังคงยืนอยู่ตรงนั้น

รอ

หรือจะไม่รอ?

(เว้นช่องว่างไว้ ไม่พูดออกมาตรงๆ)

```

## การเลือกใช้คำ

### คำศัพท์กลิ่นอายโบราณ

**เปลี่ยนคำศัพท์ยุคปัจจุบันให้เป็นคำศัพท์ย้อนยุค**

| คำปัจจุบัน | คำโบราณ/ย้อนยุค |
| --- | --- |
| เสื้อผ้า | ชายเสื้อ, ชุดยาว, กระโปรงพริ้ว

 |
| เส้นผม | ผมดำ, มวยผม

 |
| ดวงตา | แววตา, นัยน์ตา

 |
| มือ | หัตถ์, มือเรียวงาม (สตรี)

 |
| เดิน | ก้าวเดิน, ทอดน่อง, ย่างสามขุม

 |
| พูด | กล่าว, เอ่ย, เปิดปาก

 |
| หัวเราะ/ยิ้ม | ยิ้มบาง, ยิ้มเยาะ, หลุดขำ

 |
| มอง | จ้อง, มอง, ชำเลือง

 |

---

### หลีกเลี่ยงคำศัพท์ที่ดูร่วมสมัยเกินไป

#### ❌ ห้ามใช้เด็ดขาด

```
- โอเค, บ๊ายบาย, ฮ่าฮ่า
- คอมพิวเตอร์, โทรศัพท์, รถยนต์
- มาก, สุดๆ, เป็นพิเศษ (หากใช้พร่ำเพรื่อเกินไป)
- หล่อ, เท่, เจ๋ง, น่ารัก/มุ้งมิ้ง

```

#### ✅ คำโบราณทดแทน

```
- ดี, รับทราบ, อืม
- ยุทธภพ, วงการนักเลง, แผ่นดินใหญ่
- ยิ่ง, ค่อนข้าง, แสน
- สง่างาม, คมคาย, หมดจด, น่าเอ็นดู

```

## ลักษณะรูปแบบประโยค

### ประโยคสั้นได้อารมณ์กวี

**ประโยคสั้นๆ ในแนวโบราณต้องมีเสน่ห์ชวนคิด**

#### ✅ ตัวอย่างที่ 1

```
หว่านฉิงยืนกุมกระบี่นิ่ง
สายลม เริ่มพัดพาน
ชายเสื้อของนาง พริ้วไหวไปตามลม

ประกายกระบี่ เย็นเยียบราวเกล็ดน้ำค้าง

```

#### ✅ ตัวอย่างที่ 2

```
เขาหัวเราะแล้ว
เป็นรอยยิ้มที่บางเบายิ่งนัก

"ในที่สุด เจ้า ก็มา"

"อืม"

นางพยักหน้า ไม่กล่าวอะไรไปมากกว่านั้น

```

---

### ประโยคคู่ขนานและการซ้ำคำ

**ใช้โครงสร้างประโยคที่ล้อรับกันอย่างเหมาะสม เพื่อเพิ่มท่วงทำนองในการอ่าน**

#### ✅ ตัวอย่าง

```
กระบี่ อยู่ในมือ
ใจ อยู่ที่กระบี่

ตัวยังไม่ถึง กระบี่ไปก่อน
กระบวนท่าไม่ทันออก กลิ่นอายพลังพลันปรากฏ

```

**ข้อควรระวัง**:

* อย่าใช้มากเกินไป


* ใช้แค่ 2-3 ประโยคก็พอ


* เลือกใช้เฉพาะในจังหวะสำคัญ

## การออกแบบบทสนทนา

### บทสนทนาย้อนยุค

**รักษาความคลาสสิก แต่ต้องไม่พูดยากจนลิ้นพันกัน**

#### ❌ แข็งทื่อและโบราณเกินไป

```
"ท่านเป็นผู้ใด เหตุใดจึงมาสกัดกั้นเส้นทางสัญจรของข้าพเจ้า ณ ที่แห่งนี้?"
"ตัวข้านั้นมีนามว่า..."

```

#### ✅ ภาษาปัจจุบันกลิ่นอายโบราณ

```
"เจ้าเป็นใคร?"

"ลองเดาดูสิ" เขาหัวเราะ

"ไม่มีเวลาว่างมาเดา" นางยกกระบี่ขึ้น "หลีกไป"

"หากข้าไม่หลีกเล่า?"

"เช่นนั้น ก็มาวัดกันที่ฝีมือ"

```

**จุดสำคัญ**:

* สามารถใช้คำว่า "เจ้า" "ข้า" ได้ตามปกติ (ไม่จำเป็นต้องใช้คำหรูหราตลอดเวลา)


* คงคำพูดติดปากโบราณที่คุ้นเคยไว้


* บทสนทนาต้องสั้น กระชับ และทรงพลัง



---

### ระบบคำสรรพนามเรียกขาน

**เลือกใช้สรรพนามตามระดับความสัมพันธ์**

#### คนรุ่นเดียวกัน/เสมอกัน

```
- บุรุษ: สหาย, พี่ชาย, ท่านพี่
- สตรี: แม่นาง, จอมยุทธ์หญิง
- สนิทสนม: เจ้า, ข้า, เรียกชื่อตรงๆ

```

#### ผู้อาวุโสกว่า

```
- ท่านอาจารย์, อาวุโส, ผู้อาวุโส
- เจ้าสำนัก, ประมุข

```

#### ผู้เยาว์กว่า

```
- เจ้าหนู, แม่หนู, เด็กน้อย

```

## การบรรยายฉาก

### ฉากยุทธภพ

#### ✅ ตัวอย่าง: โรงเตี๊ยม

```
ภายในโรงเตี๊ยมคลาคล่ำไปด้วยผู้คน

ตรงมุมห้องมีบุรุษลึกลับคลุมหน้าผู้หนึ่ง เหล้าหนึ่งกา ตัวคนเดียว
ริมหน้าต่างมีคนกำลังเดินหมากรุก วางตัวหมากไร้เสียงสะท้อน
หลังเคาน์เตอร์ หลงจู๊กำลังดีดลูกคิดเสียงดังป๊อกแป๊กไม่ขาดสาย

หว่านฉิงผลักประตูเดินเข้ามา

ผู้คนในโรงเตี๊ยมต่างหันมามองนางแวบหนึ่ง
ก่อนจะหันกลับไปสนใจเรื่องของตนเองต่อ

นี่คือ กฎของยุทธภพ

```

---

#### ✅ ตัวอย่าง: ป่าไผ่

```
ลึกเข้าไปในป่าไผ่ มีเรือนหลังเล็กตั้งอยู่

ปูพื้นด้วยหินเขียว เงาต้นไผ่ไหวเอนตามลม
กลางลานบ้านมีคนผู้หนึ่ง กำลังร่ายรำกระบี่

ประกายกระบี่วูบวาบ พาให้เกิดสายลมพัดกรรโชกเป็นระยะ
ใบไผ่ร่วงหล่นพรู ปลิวว่อนไปตามวิถีกระบี่

กระบี่ เก็บ

ใบไผ่ ร่วงลงสู่พื้น

ทุกสิ่งกลับคืนสู่ความสงบ

เหลือเพียงสายลม ที่ยังคงพัดผ่าน

```

### ฉากต่อสู้

**ฉากต่อสู้แนวโบราณต้องเห็นภาพเคลื่อนไหวชัดเจน**

#### ✅ ตัวอย่าง

```
กระบี่ ออกจากฝัก

ประกายแสงสายหนึ่งวาบผ่าน

ฝ่ายตรงข้ามเบี่ยงกายหลบ รอดพ้นไปได้อย่างหวุดหวิด
คมกระบี่กรีดผ่านชายเสื้อของเขา จนเกิดเป็นรอยแยกแผลหนึ่ง

"กระบี่ช่างรวดเร็วนัก!"

เขายังกล่าวไม่ทันจบคำ

กระบี่ที่สอง ก็มาถึงแล้ว

คราวนี้เขาหลบไม่พ้น

คมกระบี่ หยุดนิ่งอยู่ห่างจากลำคอของเขาเพียงสามนิ้ว

"เจ้าแพ้แล้ว"

เขาแค่นยิ้มขื่น "ใช่ ข้าแพ้แล้ว"

กระบี่ คืนสู่ฝัก

```

**จุดสำคัญ**:

* ประโยคสั้นสร้างจังหวะกระชั้น


* แยกบรรทัดเดี่ยวเพื่อเพิ่มแรงกระแทกอารมณ์


* การเว้นว่าง (กระบี่ออกท่าไหน? ให้คนอ่านจินตนาการต่อเองในหัว)

## การบรรยายอารมณ์ความรู้สึก

### แสดงออกแบบนุ่มนวลเป็นนัย (การแสดงออกอย่างแยบยล)

**ความรู้สึกในแนวโบราณต้องมีความสงบเสงี่ยม ค่อยเป็นค่อยไป**

#### ❌ แสดงออกแบบทื่อๆ ตรงๆ

```
"ข้าฮักเจ้า"
"ข้าก็ฮักเจ้าเช่นกัน"

```

#### ✅ แสดงออกแบบลึกซึ้งกินใจ

```
"หากมีชาติหน้า เจ้าจะยินยอม...หรือไม่?"

นางยังกล่าวไม่จบคำ

และเขา ก็ไม่ได้ปล่อยให้นางพูดจนจบเช่นกัน

"ยินยอม"

คำเพียงคำเดียว แทนคำมั่นสัญญาตลอดกาล

```

---

### ใช้สิ่งแทนใจส่งผ่านความรู้สึก

**ใช้ธรรมชาติหรือสิ่งของรอบกายสื่อความในใจ**

#### ✅ ตัวอย่าง: พระจันทร์

```
เขาทอดสายตามองพระจันทร์

ดวงกลมโตเด่นหง้า

แต่เหตุใด ในใจกลับรู้สึกเหมือนมีบางสิ่งขาดหายไป?

นางจะกำลังมองพระจันทร์ดวงนี้อยู่เหมือนกันหรือไม่?

คงจะไม่หรอก

เพราะอย่างไรเสีย นางก็จากไปตั้งนานแล้ว

พระจันทร์ยังคงเป็นดวงเดิม
แต่คน กลับไม่ใช่คนเดิมอีกต่อไป

```

## การสร้างบรรยากาศยุคสมัย

### รายละเอียดวิถีชีวิต

**สอดแทรกรายละเอียดสิ่งของเครื่องใช้เพื่อสร้างความสมจริงของยุคสมัย**

#### ✅ ตัวอย่าง

```
นางนิ่งนั่งอยู่หน้าคันฉ่องทองเหลือง ค่อยๆ หวีผม
เส้นผมดำขลับราวน้ำตก ทิ้งตัวสลวยลงมา

สาวใช้ยกอ่างน้ำอุ่นเข้ามาภายในห้อง

"คุณหนู ได้เวลาผลัดเปลี่ยนเสื้อผ้าแล้วเจ้าค่ะ"

"อืม"

นางหยัดกายลุกขึ้น สาวใช้ช่วยนางแก้สายรัดเอว
กระโปรงยาวร่วงหล่นลงพื้น เปลี่ยนเป็นชุดรัดกุมกระฉับกระเฉงแทน

"วันนี้ไม่สวมกระโปรงแล้วหรือเจ้าคะ?"

"อืม" นางผูกสายรัดเอวให้แน่น "ต้องออกไปข้างนอก"

```

**จุดสำคัญ**:

* คันฉ่องทองเหลือง, สาวใช้, กระโปรงยาว, ชุดรัดกุม


* ล้วนเป็นรายละเอียดวิถีชีวิตโบราณ


* หลอมรวมเข้ากับการดำเนินเรื่องได้อย่างเป็นธรรมชาติ

อาภรณ์สีขาวราวหิมะ รูปร่างผอมบางแต่เหยียดตรงดูสง่างาม

สายลมพัดผ่าน พาให้ชายเสื้อของเขาพริ้วไหว

นางหยุดชะงักฝีเท้าลง

เขาเองก็เงยหน้าขึ้นมาสบตานางพอดี

สี่สายตาประสาน

ไม่มีใครเป็นฝ่ายเปิดฉากพูดก่อน

จนกระทั่งกลีบดอกแอปริคอตร่วงหล่นลงมา เกาะอยู่บนหัวไหล่ของนาง

เขาพลันยิ้ม

"แม่นาง ใต้ร่มคันนี้ ยังพอมีที่ว่างให้คนอีกสักคนหรือไม่?"

นางมองเขา นิ่งเงียบไม่พูดจา

แต่นางกลับขยับกาย เว้นพื้นที่ร่มให้ครึ่งหนึ่ง

ก้าวเดินเข้ามา ยืนเคียงข้างนาง

สายฝนยังคงโปรยปราย

พวกเขาก้าวเดินผ่านสะพานแห่งนั้นไปด้วยกันเช่นนั้นเอง

## บทสรุป

**แก่นแท้ของสไตล์ Ancient-Style**:

1. **อารมณ์โบราณความรู้สึกร่วมปัจจุบัน** - มีความงดงามคลาสสิก แต่ต้องไม่เข้าใจยากจนเกินไป


2. **ภาพจำลองต้องมาก่อน** - ใช้สิ่งของหรือธรรมชาติรอบตัวมาอธิบายความรู้สึกแทนการบรรยายทื่อๆ


3. **การแสดงออกแบบละมุนลึกซึ้ง** - ความรู้สึกต้องมีความยับยั้งชั่งใจ


4. **ศิลปะแห่งการเว้นว่าง** - ไม่พูดหมดทุกคำ ปล่อยให้คิดต่อ


5. **ท่วงทำนองแบบกวี** - ถ้อยคำภาษามีจังหวะจะโคนชวนอ่าน
